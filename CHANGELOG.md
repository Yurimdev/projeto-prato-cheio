# Changelog

Todas as mudanças notáveis deste projeto serão documentadas neste arquivo.
O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024-04-09
### Adicionado
- Projeto inicial do Projeto Prato Cheio.
- Função `add_donation` para cadastro de doações com validação de quantidades não-negativas.
- Função `list_donations` para visualização das doações registradas.
- Testes automatizados (pytest).
- Documentação no README com instruções detalhadas.
